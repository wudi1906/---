## ðŸ“‹ Recording Checklist

### Before Recording
- [ ] Start all services (.\start-all.ps1)
- [ ] Import demo data for all projects
- [ ] Test complete workflow beforehand
- [ ] Prepare script notes (don't read verbatim)
- [ ] Set up microphone and test audio
- [ ] Close distracting tabs/notifications
- [ ] Set browser zoom to 100%
- [ ] Use incognito mode (clean browser state)

### During Recording
- [ ] Speak clearly and at moderate pace
- [ ] Follow script structure
- [ ] Show features, don't just describe
- [ ] Keep mouse movements smooth
- [ ] Highlight clicks with cursor effects (if tool supports)
- [ ] Pause briefly between sections
- [ ] Smile when on camera (builds trust)

### After Recording
- [ ] Trim silent parts
- [ ] Add captions/subtitles
- [ ] Add zoom effects on key features (optional)
- [ ] Add call-to-action card at end
- [ ] Compress to <100MB
- [ ] Test playback on different devices
- [ ] Get feedback from friend/colleague
